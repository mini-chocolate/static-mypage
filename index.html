ポートフォリオサイト

〜確実な技術力でお客様の課題を解決致します〜

対応スキル一覧

HTML & CSS

Webサイトを制作する為のマークアップ言語。HTMLで文章の構造、CSSでレイアウトの構造を調整してWebサイトの制作が可能です。

BootStrap

Twitter社が開発したCSSフレームワーク。従来のCSSコーディングよりモダンなWebサイトの制作が可能になります。

Javascript

動的なWebサイトを制作するにあたって必要なプログラミング言語。更にJQueryなどのライブラリを利用してブラウザに依存しないWebサイト制作が可能になります。

WordPress

全世界のWebサイトの3割以上に利用されているCMS。プラグインを利用してモダンなサイトや動的なサイトの制作まで提供する事が可能になります。

PHP

掲示板や問い合わせフォームなどに利用されるサーバサイドプログラミング言語。WordPressのプラグインもPHPで作成されている為、独自にテンプレートをカスタマイズを行う事も可能です。

MySQL

データを蓄積・管理するリレーショナルデータベース管理システム。

Linux

Webサイトの土台となるWebサーバやメールの送受信に必要になるメールサーバを構築する際に利用するOS。日本ではCentOSをメインにUbuntuなども利用されています。

AWS

アマゾン社が提供するクラウドサービス。オンプレミスに囚われずクラウド上で自在にインフラ基盤の提供を行う事が可能です。

提供サービス一覧

WEBサイト制作

ご要望となるHTML・CSSコーディング・Javascript・WordPressを利用したWebサイト制作を行います。またメンテナンスのし易い可読性の高いコーディングやSEO対策に至るまでプロフェッショナルとしてしっかりと責任を持って対応させて頂きます。

インフラ構築

レンタルサーバ・VPS・クラウドサービスを利用してお客様が望むインフラ環境を構築致します。

提供プラン一覧

Webサイト制作プラン

既にサーバ・ドメインを取得しており、Webサイト制作をご希望の方はこちらのプランになります。お客様のご要望に沿ったWebサイト制作をプロフェショナルとして責任を持って対応させて頂きます。またフロントエンドに限らずバックエンド処理にも対応致しますので、ご遠慮なくお問い合わせ下さい。

インフラ構築プラン

Web制作は他の人に依頼する物の、必要なサーバの調達などのインフラ構築・保守をご希望の方はこちらのプランになります。お客様のご希望に沿った信頼性のあるインフラ提供をプロフェショナルとして責任を持って対応させて頂きます。

全てお任せプラン

Webサイトを動作させるにあたって必要なインフラの構築からWeb制作まで一貫して対応をご希望の方はこちらのプランになります。基本的にお客様の要望に応じてこちら側で全ての制作を対応致しますので、ご安心してご利用頂けるプランとなっております。
